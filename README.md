# Performance Testing Framework

A containerized performance testing framework built with **k6**, **Java utilities**, **Prometheus**, and **Grafana**, designed for scalable and reproducible load testing.

It provides a full observability stack for executing performance tests, collecting metrics, and visualizing system behavior under load.

---

# Project Overview

This framework combines multiple layers of performance testing and observability:

- **k6** → Load testing engine (JavaScript-based scenarios)
- **Prometheus** → Metrics collection and time-series storage
- **Grafana** → Visualization and dashboards
- **Docker Compose** → Full environment orchestration
